# Usage
<!-- usage -->
```sh-session
$ npm install -g pear
$ pear COMMAND
running command...
$ pear (-v|--version|version)
pear/0.0.0 darwin-x64 node-v11.3.0
$ pear --help [COMMAND]
USAGE
  $ pear COMMAND
...
```
